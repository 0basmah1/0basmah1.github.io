Custom Encryption Algorithm

The project implements a custom encryption algorithm using SHA-256 hashing for integrity verification, combined with binary conversion, bitwise complement, 3-bit rotation, and XOR operations to ensure confidentiality and secure data transmission.
For decryption, these steps are applied in reverse order, allowing the original plaintext to be restored.
